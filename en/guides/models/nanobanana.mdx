## Overview

nanobanana is a fast, efficient image generation model for general use cases.

- **Type**: Image Generation
- **Processing**: Asynchronous
- **Resolution**: Up to 2048x2048
- **Price**: $0.01 per image

## Parameters

<ParamField body="model" type="string" required>
  `nanobanana`
</ParamField>

<ParamField body="prompt" type="string" required>
  Description of the image to generate (max 2000 characters)
</ParamField>

<ParamField body="negative_prompt" type="string">
  What to avoid in the image (max 1000 characters)
</ParamField>

<ParamField body="width" type="integer">
  Image width in pixels (256-2048, default: 1024)
</ParamField>

<ParamField body="height" type="integer">
  Image height in pixels (256-2048, default: 1024)
</ParamField>

<ParamField body="num_outputs" type="integer">
  Number of images to generate (1-4, default: 1)
</ParamField>

<ParamField body="image_url" type="string">
  Reference image URL for image-to-image generation
</ParamField>

<ParamField body="webhook_url" type="string">
  Webhook callback URL for task completion or failure notifications
</ParamField>

## Example

<CodeGroup>

```bash cURL
curl -X POST https://api.eggapi.ai/v1/generate \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "nanobanana",
    "prompt": "A futuristic cityscape at sunset",
    "width": 1024,
    "height": 1024,
    "num_outputs": 1
  }'
```

```python Python
import requests

response = requests.post(
    "https://api.eggapi.ai/v1/generate",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "model": "nanobanana",
        "prompt": "A futuristic cityscape at sunset",
        "width": 1024,
        "height": 1024,
        "num_outputs": 1
    }
)

task = response.json()["data"]
print(f"Task ID: {task['id']}")
```

```javascript JavaScript
const response = await fetch("https://api.eggapi.ai/v1/generate", {
  method: "POST",
  headers: {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
  },
  body: JSON.stringify({
    model: "nanobanana",
    prompt: "A futuristic cityscape at sunset",
    width: 1024,
    height: 1024,
    num_outputs: 1
  })
});

const { data: task } = await response.json();
console.log("Task ID:", task.id);
```
