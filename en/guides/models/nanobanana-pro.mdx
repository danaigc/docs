## Overview

nanobanana-pro is a professional-grade image generation model with enhanced detail.

- **Type**: Image Generation
- **Processing**: Asynchronous
- **Aspect ratio**: 16:9 or 9:16
- **Price**: $0.016 per image

## Parameters

<ParamField body="model" type="string" required>
  `nanobanana-pro`
</ParamField>

<ParamField body="prompt" type="string" required>
  Description of the image to generate (max 2000 characters)
</ParamField>

<ParamField body="negative_prompt" type="string">
  What to avoid in the image (max 1000 characters)
</ParamField>

<ParamField body="aspect_ratio" type="string">
  Aspect ratio: `16:9`, `9:16` (default: `16:9`)
</ParamField>

<ParamField body="num_outputs" type="integer">
  Number of images to generate (1-4, default: 1)
</ParamField>

<ParamField body="image_url" type="string">
  Reference image URL for image-to-image generation
</ParamField>

<ParamField body="webhook_url" type="string">
  Webhook callback URL for task completion or failure notifications
</ParamField>

## Example

<CodeGroup>

```bash cURL
curl -X POST https://api.eggapi.ai/v1/generate \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "nanobanana-pro",
    "prompt": "A futuristic cityscape at night, cyberpunk style",
    "negative_prompt": "blurry, low quality",
    "aspect_ratio": "16:9",
    "num_outputs": 2,
    "webhook_url": "https://webhook.site/your-webhook-id"
  }'
```

```python Python
import requests

response = requests.post(
    "https://api.eggapi.ai/v1/generate",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "model": "nanobanana-pro",
        "prompt": "A futuristic cityscape at night, cyberpunk style",
        "negative_prompt": "blurry, low quality",
        "aspect_ratio": "16:9",
        "num_outputs": 2,
        "webhook_url": "https://webhook.site/your-webhook-id"
    }
)

task = response.json()["data"]
print(f"Task ID: {task['id']}")
```

## Image Editing Mode

nanobanana-pro also supports image editing, allowing you to modify existing images using natural language prompts with professional-grade quality.

- **Endpoint**: `POST /flow/v1/images/edits`
- **Content Type**: `multipart/form-data`
- **Max Images**: 9 images per request
- **Processing**: Asynchronous

### Image Editing Parameters

<ParamField body="model" type="string" required>
  `nano_banana_pro` (Note: use underscores for image editing)
</ParamField>

<ParamField body="prompt" type="string" required>
  Description of how to edit the image (max 2000 characters)
</ParamField>

<ParamField body="image[]" type="file[]">
  Image files to edit (supports up to 9 images). Use this OR `image_url[]`.
</ParamField>

<ParamField body="image_url[]" type="string[]">
  URLs of images to edit (supports up to 9 URLs). Use this OR `image[]`.
</ParamField>

<ParamField body="webhook_url" type="string">
  Webhook callback URL for task completion or failure notifications
</ParamField>

### Image Editing Example

<CodeGroup>

```bash cURL (File Upload)
curl -X POST https://api.eggapi.ai/flow/v1/images/edits \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -F "image[]=@input.jpg" \
  -F "prompt=Transform into a professional studio portrait" \
  -F "model=nano_banana_pro"
```

```bash cURL (URL Input)
curl -X POST https://api.eggapi.ai/flow/v1/images/edits \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -F "image_url[]=https://example.com/image.jpg" \
  -F "prompt=Transform into a professional studio portrait" \
  -F "model=nano_banana_pro"
```

```python Python
import requests

# Using file upload
with open("input.jpg", "rb") as f:
    response = requests.post(
        "https://api.eggapi.ai/flow/v1/images/edits",
        headers={"Authorization": "Bearer YOUR_API_KEY"},
        files=[("image[]", f)],
        data={
            "prompt": "Transform into a professional studio portrait",
            "model": "nano_banana_pro"
        }
    )

task = response.json()["data"]
print(f"Task ID: {task['id']}")
```

