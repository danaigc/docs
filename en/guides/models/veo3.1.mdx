## Overview

veo3.1 is a video generation model with excellent motion consistency.

- **Type**: Video Generation
- **Processing**: Asynchronous
- **Duration**: 1-60 seconds
- **Price**: $0.08 per second

## Parameters

<ParamField body="model" type="string" required>
  `veo3.1`
</ParamField>

<ParamField body="prompt" type="string" required>
  Description of the video to generate (max 2000 characters)
</ParamField>

<ParamField body="negative_prompt" type="string">
  What to avoid in the video (max 1000 characters)
</ParamField>

<ParamField body="duration" type="integer">
  Video duration in seconds (1-60, default: 5)
</ParamField>

<ParamField body="aspect_ratio" type="string">
  Aspect ratio: `16:9`, `9:16`, `1:1`, `4:3`, `3:4`
</ParamField>

<ParamField body="image_url" type="string">
  Reference image URL for image-to-video generation
</ParamField>

<ParamField body="webhook_url" type="string">
  Webhook callback URL for task completion or failure notifications
</ParamField>

## Example

<CodeGroup>

```bash cURL
curl -X POST https://api.eggapi.ai/v1/generate \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "veo3.1",
    "prompt": "A timelapse of clouds moving over mountains",
    "duration": 10,
    "aspect_ratio": "16:9"
  }'
```

```python Python
import requests

response = requests.post(
    "https://api.eggapi.ai/v1/generate",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "model": "veo3.1",
        "prompt": "A timelapse of clouds moving over mountains",
        "duration": 10,
        "aspect_ratio": "16:9"
    }
)

task = response.json()["data"]
print(f"Task ID: {task['id']}")
```

```javascript JavaScript
const response = await fetch("https://api.eggapi.ai/v1/generate", {
  method: "POST",
  headers: {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
  },
  body: JSON.stringify({
    model: "veo3.1",
    prompt: "A timelapse of clouds moving over mountains",
    duration: 10,
    aspect_ratio: "16:9"
  })
});

const { data: task } = await response.json();
console.log("Task ID:", task.id);
```
