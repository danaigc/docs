## Prerequisites

Before you begin, you'll need:

1. An Egg API account - [Sign up here](https://eggapi.ai)
2. An API key from your [dashboard](https://eggapi.ai/dashboard)

## Step 1: Get Your API Key

<Steps>
  <Step title="Sign up or log in">
    Go to [eggapi.ai](https://eggapi.ai) and create an account or log in.
  </Step>
  <Step title="Navigate to API Keys">
    In your dashboard, go to the API Keys section.
  </Step>
  <Step title="Create a new key">
    Click "Create API Key" and give it a name. Copy and save your key securely - you won't be able to see it again.
  </Step>
</Steps>

<Warning>
  Keep your API key secure. Never expose it in client-side code or public repositories.
</Warning>

## Step 2: Make Your First Request

### Generate an Image (Asynchronous)

当前所有模型均为异步生成。接口会返回任务 ID，需轮询获取结果。你也可以传入 `webhook_url`，并使用 `https://webhook.site/` 获取测试回调地址。

<CodeGroup>

```bash cURL
curl -X POST https://api.eggapi.ai/v1/generate \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "nanobanana-pro",
    "prompt": "A cute robot holding a coffee cup, digital art style",
    "aspect_ratio": "16:9",
    "webhook_url": "https://webhook.site/your-webhook-id"
  }'
```

```python Python
import requests

response = requests.post(
    "https://api.eggapi.ai/v1/generate",
    headers={
        "Authorization": "Bearer YOUR_API_KEY",
        "Content-Type": "application/json"
    },
    json={
        "model": "nanobanana-pro",
        "prompt": "A cute robot holding a coffee cup, digital art style",
        "aspect_ratio": "16:9",
        "webhook_url": "https://webhook.site/your-webhook-id"
    }
)

task = response.json()["data"]
print(f"Task ID: {task['id']}")
```

```javascript JavaScript
const response = await fetch("https://api.eggapi.ai/v1/generate", {
  method: "POST",
  headers: {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
  },
  body: JSON.stringify({

    prompt: "A cute robot holding a coffee cup, digital art style",

    webhook_url: "https://webhook.site/your-webhook-id"
  })
});

const { data: task } = await response.json();
console.log("Task ID:", task.id);
```

### Poll for Results

Use the task ID from any model (image or video) to poll for results:

<CodeGroup>

```bash cURL
# Step 1: Submit the task
curl -X POST https://api.eggapi.ai/v1/generate \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "nanobanana-pro",
    "prompt": "A cat playing with a ball of yarn",
    "aspect_ratio": "16:9",
    "webhook_url": "https://webhook.site/your-webhook-id"
  }'

# Response: {"data": {"id": "gen_xyz789", "status": "pending", ...}}

# Step 2: Poll for results
curl https://api.eggapi.ai/v1/tasks/gen_xyz789 \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```python Python
import requests
import time

# Step 1: Submit the task
response = requests.post(
    "https://api.eggapi.ai/v1/generate",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "model": "nanobanana-pro",
        "prompt": "A cat playing with a ball of yarn",
        "aspect_ratio": "16:9",
        "webhook_url": "https://webhook.site/your-webhook-id"
    }
)

task = response.json()["data"]
task_id = task["id"]
print(f"Task created: {task_id}")

# Step 2: Poll for results
while True:
    status_response = requests.get(
        f"https://api.eggapi.ai/v1/tasks/{task_id}",
        headers={"Authorization": "Bearer YOUR_API_KEY"}
    )
    task = status_response.json()["data"]

    if task["status"] == "completed":
        print(f"Image URL: {task['output']['url']}")
        print(f"Cost: {task['cost']}")
        break
    elif task["status"] == "failed":
        print(f"Error: {task['error']}")
        break
    else:
        print(f"Status: {task['status']}...")
        time.sleep(5)
```

</CodeGroup>
