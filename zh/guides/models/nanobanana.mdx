## Overview

nanobanana is a fast, efficient image generation model for general use cases.

- **Type**: Image Generation
- **Processing**: Asynchronous
- **Aspect ratio**: 16:9 or 9:16
- **Price**: $0.01 per image

## Parameters

<ParamField body="model" type="string" required>
  `nanobanana`
</ParamField>

<ParamField body="prompt" type="string" required>
  Description of the image to generate (max 2000 characters)
</ParamField>

<ParamField body="negative_prompt" type="string">
  What to avoid in the image (max 1000 characters)
</ParamField>

<ParamField body="aspect_ratio" type="string">
  Aspect ratio: `16:9`, `9:16` (default: `16:9`)
</ParamField>

<ParamField body="num_outputs" type="integer">
  Number of images to generate (1-4, default: 1)
</ParamField>

<ParamField body="image_urls" type="string[]">
  用于图生图编辑的图片 URL 数组（最多支持 9 张）
</ParamField>

<ParamField body="webhook_url" type="string">
  Webhook 回调地址，用于接收任务完成或失败通知
</ParamField>

## Example

<CodeGroup>

```bash cURL
curl -X POST https://api.eggapi.ai/v1/generate \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "nanobanana",
    "prompt": "A futuristic cityscape at sunset",
    "aspect_ratio": "16:9",
    "num_outputs": 1,
    "webhook_url": "https://webhook.site/your-webhook-id"
  }'
```

```python Python
import requests

response = requests.post(
    "https://api.eggapi.ai/v1/generate",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "model": "nanobanana",
        "prompt": "A futuristic cityscape at sunset",
        "aspect_ratio": "16:9",
        "num_outputs": 1,
        "webhook_url": "https://webhook.site/your-webhook-id"
    }
)

task = response.json()["data"]
print(f"Task ID: {task['id']}")
```

```javascript JavaScript
const response = await fetch("https://api.eggapi.ai/v1/generate", {
  method: "POST",
  headers: {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
  },
  body: JSON.stringify({
    model: "nanobanana",
    prompt: "A futuristic cityscape at sunset",
    aspect_ratio: "16:9",
    num_outputs: 1,
    webhook_url: "https://webhook.site/your-webhook-id"
  })
});

const { data: task } = await response.json();
console.log("Task ID:", task.id);
```

## 图片编辑模式

nanobanana 还支持图片编辑功能，允许您使用自然语言提示修改现有图片。只需在请求中提供 `image_urls` 参数即可。

- **端点**: `POST /v1/generate`（与图片生成相同）
- **内容类型**: `application/json`
- **最大图片数**: 每次请求最多 9 张图片
- **处理方式**: 异步

### 图片编辑参数

<ParamField body="model" type="string" required>
  `nanobanana`
</ParamField>

<ParamField body="prompt" type="string" required>
  描述如何编辑图片（最多 2000 字符）
</ParamField>

<ParamField body="image_urls" type="string[]" required>
  要编辑的图片 URL 数组（最多 9 个）
</ParamField>

<ParamField body="webhook_url" type="string">
  Webhook 回调地址，用于接收任务完成或失败通知
</ParamField>
